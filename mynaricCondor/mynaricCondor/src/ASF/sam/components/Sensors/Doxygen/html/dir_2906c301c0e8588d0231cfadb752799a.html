Detailed Description

Overview

The stdout_ide_tests_report replaces the normal ceedling "pretty" output with a simplified variant intended to be easily parseable.

Setup

Enable the plugin in your project.yml by adding stdout_ide_tests_report to the list of enabled plugins.

YAML
:plugins:
:enabled:
- stdout_ide_tests_report
