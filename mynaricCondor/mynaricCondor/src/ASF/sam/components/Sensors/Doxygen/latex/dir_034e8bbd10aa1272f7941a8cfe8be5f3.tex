\hypertarget{README.md_autotoc_md0}{}\doxysubsection{\texorpdfstring{Plugin Overview}{Plugin Overview}}\label{README.md_autotoc_md0}
Plugin for integrating Bullseye code coverage tool into Ceedling projects. This plugin requires a working license to Bullseye code coverage tools. The tools must be within the path or the path should be added to the environment in the {\ttfamily project.\+yml file}.\hypertarget{README.md_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{Configuration}{Configuration}}\label{README.md_autotoc_md1}
